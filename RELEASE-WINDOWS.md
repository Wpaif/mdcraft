# mdcraft - Release Windows x64

## Como usar

1. Baixe o arquivo `mdcraft-windows-x64.zip` e extraia o conteúdo.
2. Execute o arquivo `mdcraft.exe`.

## Requisitos

- Windows 7, 8, 10 ou 11 (64 bits)
- Pode ser necessário instalar as DLLs do MinGW caso o programa acuse falta de dependências. As principais são:
  - `libgcc_s_seh-1.dll`
  - `libstdc++-6.dll`
  - `libwinpthread-1.dll`

Essas DLLs podem ser encontradas facilmente pesquisando no Google ou instalando o [MinGW-w64](http://mingw-w64.org/).

## Observações

- O programa foi testado no Windows e via Wine.
- Caso encontre algum problema, reporte na página do projeto.
